# VANDERER — Project Page

Source for the [VANDERER](https://github.com/) project website:
**Map-Free Exploration using Future-Aware and Visual-Curiosity-Guided Diffusion Policy.**

Static site (HTML + Bulma CSS) based on the
[Nerfies](https://github.com/nerfies/nerfies.github.io) template.

## Local preview
```bash
python3 -m http.server 8000
# open http://localhost:8000
```

## Deploy (GitHub Pages)
Push to `main`, then enable **Settings → Pages → Deploy from a branch → main / root**.
Live at `https://<github-username>.github.io/vanderer`.
